---
description: "애플리케이션 개발 서버 실행 (옵션: 프로덕션 서버 확인)"
---

1. **개발 서버 실행 (Development)**:
   - `nodemon`을 사용하여 파일 변경 시 자동 재시작되는 개발 서버를 구동합니다.
   
   ```bash
   npm run dev
   ```

2. **프로덕션 서버 실행 (Production)**:
   - 운영 환경으로 서버를 구동합니다.
   
   ```bash
   npm start
   ```

3. **실행 확인**:
   - `http://localhost:3014` 로 접속하여 정상적으로 서버가 구동되었는지 확인합니다.
