# Agent Terminal Rules (에이전트 터미널 규칙)

AI 에이전트 환경에서 터미널 명령어 실행 시 발생하는 펜딩 현상(멈춤 현상)과 오류를 방지하기 위해 다음 규칙을 반드시 준수해야 합니다.

## 1. 비동기 타임아웃 및 백그라운드 실행 (Async & Background)
"실행이 오래 걸리거나 응답을 반환하지 않는 명령어(예: 데몬 실행, 워치 모드 등)는 실행 전 명시적으로 백그라운드 실행 의도를 밝히거나, 파일 출력으로 리다이렉션(> log.txt &) 처리하십시오."

## 2. 사용자 상호작용 및 Pager 사용 금지 (No Interactive Pagers)
- 명령어 실행 시 사용자 상호작용을 유발하는 Pager(`less`, `vi` 등) 사용을 엄격히 금지합니다.
- `git log`, `git diff` 등의 명령어는 항상 `--no-pager` 옵션을 사용하십시오.
